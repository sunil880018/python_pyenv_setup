---------------------------- virtual environment setup--------------------

1) pyenv install 3.8.5 // select your version in your project,install specific python version
2) pyenv local 3.8.5 // for your local at project level python version
3) pyenv virtualenv <environment_name> // creating virtual environment
   for ex- pyenv virtualenv 3.7.4 order-micro
4) pyenv activate <environment_name> // for activating your environment
5) pyenv deactivate <environment_name> // deactivate your environment
6) pyenv virtualenvs // list of virtualenvs
7) pyenv --help
